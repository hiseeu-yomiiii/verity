# 衡准 · Verity

项目材料 AI 质检与评审工具

## 定位

帮助大学生竞赛团队在提交项目材料前，
通过 AI 模拟评审发现：

- 逻辑问题
- 创新风险
- 证据缺口
- 表达问题

## 技术

Frontend:
HTML/CSS/JavaScript

AI:
Cloudflare Worker
DeepSeek API

## Features

- 项目材料分析
- 五维评分
- 评委追问模拟
- 修改行动清单
